## 📝 Visión del Proyecto

Estudio+ es una aplicación móvil diseñada para estudiantes universitarios que busca centralizar y optimizar la gestión de su vida académica. El objetivo es proporcionar una herramienta todo-en-uno para organizar horarios, tareas, exámenes y materiales de estudio.

## 🎯 Usuarios Objetivo

*   Estudiantes universitarios
*   Estudiantes de educación media

## ✨ Propuesta de Valor

*   **Organización centralizada:** Gestiona asignaturas, tareas y exámenes en un solo lugar.
*   **Recordatorios inteligentes:** Nunca olvides una fecha de entrega importante.
*   **Mapa del campus:** Encuentra tus aulas y laboratorios fácilmente.
*   **Modo offline:** Accede a tus apuntes y notas sin conexión a internet.
*   **Colaboración:** Comparte tus apuntes con compañeros al instante.

## 🛠️ Tecnologías Utilizadas

*   **Framework:** [Ionic Framework](https://ionicframework.com/) + [Angular](https://angular.io/)
*   **Plataforma Nativa:** [Capacitor](https://capacitorjs.com/)
*   **Base de datos Local:** SQLite / Ionic Storage
*   **Control de Versiones:** Git y GitHub
